# 项目简介
为了降低自然灾害带来的影响，快速的恢复生活生产，开发出了区域性应急物资管理系统，包含应急物资的存储、运输、捐赠、审批、以及用户权限的划分。
# 技术选型
## 系统架构
本系统采用分布式微服务架构，后端整体使用springCloudAlibaba开源框架。
采用前后端分离开发。
使用nginx做负载均衡。
